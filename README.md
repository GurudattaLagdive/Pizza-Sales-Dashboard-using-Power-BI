# 🍕 Pizza Sales Dashboard (Power BI)

## 📌 Project Overview

This project presents an interactive Pizza Sales Dashboard built using Power BI. The dashboard provides insights into sales performance, order trends, best-selling pizzas, revenue analysis, and customer purchasing patterns.

## 📊 Key Metrics

* Total Revenue
* Total Orders
* Average Order Value
* Total Pizzas Sold
* Daily and Monthly Sales Trends
* Best & Worst Selling Pizzas
* Category-wise Performance

## 🛠 Tools Used

* Power BI Desktop
* DAX
* Power Query
* CSV Dataset
* GitHub
)

## 📁 Files Included

* `Pizza_Sales.pbix` – Power BI report file
* `pizza_sales.csv` – Dataset
* `dashboard_screenshot.png` – Dashboard screenshot

## 🚀 How to Use

1. Download the repository.
2. Open `Pizza_Sales.pbix` in Power BI Desktop.
3. Refresh the data source if required.
4. Explore the interactive dashboard.
